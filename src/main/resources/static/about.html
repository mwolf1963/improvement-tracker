About The Application

This was developed as a learning exercise for Spring Boot using Java. The concept was to use a web application to track improvement in a business environment, keeping the learnings separate, yet searchable and trackable over time. The current application uses SQLite to track the improvements (for simplicity and to make sure Spring and Heroku were the focus of my project).
